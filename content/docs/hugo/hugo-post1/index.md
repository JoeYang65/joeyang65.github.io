# 欢迎使用Hugo框架
Hugo是世界上最快的网站构建框架。


Hugo 是最受欢迎的开源静态网站生成器之一。凭借其惊人的速度和灵活性，Hugo 让搭建网站重新变得有趣。


## 开源
Hugo 是开源且免费使用的。它采用 Apache 2.0 许可证发布。
- 速度优化：用 Go 编写，优化速度，设计灵活。凭借先进的模板系统和快速的资源流水线，Hugo 能在几秒钟内渲染一个大型网站，通常更短。
- 灵活框架：凭借其多语言支持和强大的分类系统，Hugo 被广泛用于创建文档网站、着陆页、企业、政府、非营利组织、教育、新闻、活动和项目网站。
- 快速资产管道：图像处理（转换、调整大小、裁剪、旋转、调整颜色、应用滤镜、叠加文本和图片、提取 EXIF 数据）、JavaScript 捆绑（树状摇动、代码拆分）、Sass 处理，以及出色的 TailwindCSS 支持。
- 嵌入式网页服务器：在开发过程中使用 Hugo 嵌入式网页服务器，可以即时看到内容、结构、行为和展示的变化。
