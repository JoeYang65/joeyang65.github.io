## 概述
你是否厌倦了手动管理你的 markdown 文章？你想简化内容创作和管理流程吗？不用再找了，看看 Visual Studio Code 的 Front Matter 扩展。

Front Matter 为开发者和内容创作者设计，赋予你完整的内容管理系统（CMS）的强大控制权，无需繁琐地处理服务器、网站或 API 的烦恼。通过 Front Matter，你可以直接在 Visual Studio Code 中使用你喜欢的工具和工作流程创建和管理内容。

该扩展支持多种静态站点生成器和框架，包括 Astro、Hugo、Jekyll、Hexo、NextJs、Gatsby 等。

## 特点
Front Matter 配备了一系列强大的功能，包括：

- 一个页面仪表盘，方便你搜索、筛选和输入内容。
- 网站预览，能在发布前准确展示内容的呈现效果。
- SEO 检查帮助你优化内容以适应搜索引擎。
- 支持自定义动作/脚本。
- 还有更多功能。
